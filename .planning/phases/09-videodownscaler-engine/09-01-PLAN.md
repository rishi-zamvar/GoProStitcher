Implementation steps inside `downscale(url:outputName:progress:)`:

1. **Find ffmpeg:** Check ["/opt/homebrew/bin/ffmpeg", "/usr/local/bin/ffmpeg", "/usr/bin/ffmpeg"] via FileManager.default.fileExists. Throw `VideoDownscalerError.ffmpegNotFound` immediately if none found.

2. **Validate input exists:** `guard FileManager.default.fileExists(atPath: url.path)` else throw `VideoDownscalerError.inputNotFound(url)`.

3. **Resolution guard via ffprobe:** Call `probeResolution(ffmpegPath:url:)` (private helper below). If result height is non-nil and height <= 1080, throw `VideoDownscalerError.alreadyAtTargetResolution`. (If ffprobe is unavailable or returns nil, skip the guard and proceed — do not block on probe failure.)

4. **Compute collision-free output path:** Place output in the same directory as the source. The `outputName` parameter is the desired base filename (e.g. "clip_1080p.mp4"). Strip any path components from it (use `.lastPathComponent`). Then call `collisionFreeURL(in: sourceDir, name: outputName)` — same loop pattern as AudioExtractor: try `name` first, then insert `_1`, `_2`, etc. before the extension up to 999. Specifically: split `outputName` into stem and ext at the last `.`; try `"\(stem).\(ext)"`, then `"\(stem)_1.\(ext)"`, etc.

5. **Set up defer for partial cleanup:** `var cleanupOnFailure = true; defer { if cleanupOnFailure { try? FileManager.default.removeItem(at: outputURL) } }`. Set `cleanupOnFailure = false` just before returning the URL.

6. **Probe total duration:** Call `probeDuration(ffmpegPath:url:)` — identical to AudioExtractor's private helper (derive ffprobe path by replacing "ffmpeg" with "ffprobe" in the found path).

7. **Run ffmpeg:** Create a Process with:
   - executableURL: URL(fileURLWithPath: ffmpeg)
   - arguments: `["-y", "-i", url.path, "-vf", "scale=-2:1080", "-c:v", "libx264", "-preset", "slow", "-crf", "18", "-c:a", "copy", "-progress", "pipe:1", outputURL.path]`
   - standardError: FileHandle.nullDevice
   - standardOutput: set to a Pipe when progress callback is provided

8. **Parse progress from pipe:** When `progress != nil` and `totalDuration != nil && totalDuration! > 0`, attach `pipe.fileHandleForReading.readabilityHandler`. Parse lines from the pipe data:
   - `out_time_us=<value>` → secondsProcessed = Double(value)! / 1_000_000
   - `bitrate=<value>kbits/s` → bitrateKbps = Double(value) (strip "kbits/s" suffix; handle "N/A" by using 0)
   - `fps=<value>` → fps = Double(value) ?? 0
   - Accumulate these three values across lines within a single chunk. When `out_time_us` is seen, fire the callback with a DownscaleProgress built from the accumulated values and computed fraction = min(secondsProcessed / totalDuration, 1.0).
   - When progress is nil, set process.standardOutput = FileHandle.nullDevice instead of a Pipe.

9. **Run and wait:** `try process.run(); process.waitUntilExit()`. Clear the readabilityHandler after wait.

10. **Check exit code:** `guard process.terminationStatus == 0` else throw `VideoDownscalerError.encodingFailed("exit code \(process.terminationStatus)")`.

11. **Return:** Set `cleanupOnFailure = false; return outputURL`.

**Private helpers to add:**

`probeResolution(ffmpegPath: String, url: URL) -> Int?` — returns the video height:
- Derive ffprobePath by replacing "ffmpeg" with "ffprobe"
- Run ffprobe with args: `["-v", "error", "-select_streams", "v:0", "-show_entries", "stream=height", "-of", "default=noprint_wrappers=1:nokey=1", url.path]`
- Parse stdout as Int and return it. Return nil on any failure.

`probeDuration(ffmpegPath: String, url: URL) -> Double?` — identical to AudioExtractor's private probeDuration helper.

`collisionFreeURL(in directory: URL, name: String) throws -> URL` — same pattern as AudioExtractor's collisionFreeURL but takes the full name (stem + ext) instead of stem and ext separately:
- Split name into stem and ext at last ".": `let parts = name.components(separatedBy: "."); let ext = parts.last ?? "mp4"; let stem = parts.dropLast().joined(separator: ".")`
- Try base path first; loop with _1, _2 up to _999; throw `VideoDownscalerError.outputWriteFailed(...)` if all exhausted.

/// Generates a short synthetic 4K (3840x2160) MP4 using ffmpeg lavfi.
/// Duration: 2 seconds. Audio: sine tone. Returns URL of created file.
private func make4KFixture(named name: String) throws -> URL {
    guard let ffmpeg = ffmpegPath else {
        throw XCTSkip("ffmpeg not available")
    }
    let dest = tempDir.appendingPathComponent(name)
    let process = Process()
    process.executableURL = URL(fileURLWithPath: ffmpeg)
    process.arguments = [
        "-y",
        "-f", "lavfi", "-i", "color=c=blue:size=3840x2160:rate=30",
        "-f", "lavfi", "-i", "sine=frequency=440:duration=2",
        "-t", "2",
        "-c:v", "libx264", "-preset", "ultrafast", "-crf", "35",
        "-c:a", "aac",
        dest.path
    ]
    process.standardOutput = FileHandle.nullDevice
    process.standardError = FileHandle.nullDevice
    try process.run()
    process.waitUntilExit()
    guard process.terminationStatus == 0 else {
        throw NSError(domain: "test", code: 1,
                      userInfo: [NSLocalizedDescriptionKey: "lavfi fixture generation failed"])
    }
    return dest
}
