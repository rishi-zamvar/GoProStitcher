  <behavior>
    ChunkReviewFeature TCA @Reducer state:

    ```swift
    @ObservableState
    struct State: Equatable {
        var chunks: [ScannedChunk]          // mutable for drag-reorder
        var selectedPreviewURL: URL? = nil  // non-nil = preview modal shown
        var metadata: [URL: ChunkMetadata] = [:] // loaded async per chunk
    }

    struct ChunkMetadata: Equatable {
        let duration: TimeInterval
        let resolution: CGSize
        let thumbnail: NSImage?
    }
    ```

    Actions:
    - `chunksReordered(from: IndexSet, to: Int)` — applies Array.move(fromOffsets:toOffset:) on state.chunks
    - `chunkTapped(URL)` — sets selectedPreviewURL to the tapped URL
    - `previewDismissed` — sets selectedPreviewURL to nil
    - `loadMetadata(URL)` — fires .run effect calling AVMetadataReader.duration + resolution + thumbnail, then sends metadataLoaded
    - `metadataLoaded(URL, ChunkMetadata)` — stores in state.metadata dictionary
    - `loadAllMetadata` — fires loadMetadata for each chunk URL (called on appear)

    Reducer logic:
    - chunksReordered: state.chunks.move(fromOffsets: from, toOffset: to); return .none
    - chunkTapped: state.selectedPreviewURL = url; return .none
    - previewDismissed: state.selectedPreviewURL = nil; return .none
    - loadMetadata: .run { [url] send in let d = await AVMetadataReader.duration(url:url) ?? 0; let r = await AVMetadataReader.resolution(url:url) ?? .zero; let t = await AVMetadataReader.thumbnail(url:url); await send(.metadataLoaded(url, ChunkMetadata(duration:d, resolution:r, thumbnail:t))) }
    - metadataLoaded: state.metadata[url] = meta; return .none
    - loadAllMetadata: .merge(state.chunks.map { .send(.loadMetadata($0.url)) })

    Note: ChunkMetadata.thumbnail is NSImage? — NSImage does not conform to Equatable.
    Two options (pick one and be consistent):
    Option A: Make ChunkMetadata.Equatable ignore thumbnail: static func == compares duration and resolution only.
    Option B: Store thumbnail separately in state as [URL: NSImage] outside ChunkMetadata so ChunkMetadata remains fully Equatable.
    Use Option A (simpler — thumbnail equality is not needed for business logic).

    Reorder test cases:
    - move index 2 to 0: [A,B,C] -> [C,A,B]
    - move index 0 to end: [A,B,C] -> [B,C,A]
    - move middle: [A,B,C,D] move {1} to 3 -> [A,C,B,D] (Array.move semantics match SwiftUI List onMove)

    Preview state tests:
    - chunkTapped(urlA) -> selectedPreviewURL == urlA
    - previewDismissed after tap -> selectedPreviewURL == nil
