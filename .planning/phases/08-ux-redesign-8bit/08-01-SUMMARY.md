## Decisions Made
- **ATSApplicationFontsPath = "."** — xcodegen's `buildPhase: resources` on a folder copies contents flat into `Resources/`, so `Fonts` subdirectory never exists in the built bundle. Using `.` registers all fonts from the Resources root. Verified by CTFont PostScript name lookup returning "JetBrainsMono-Regular"/"JetBrainsMono-Bold" (not "Helvetica").
- **displayString as internal computed property** — RetroProgressBar exposes the block string computation separately from `body`. Tests call `bar.displayString` directly without needing ViewInspector or SwiftUI rendering infrastructure.
- **Color(hex:) uses explicit RGB init** — ensures sRGB color space (not device RGB), allowing `NSColor.usingColorSpace(.sRGB)` in tests to get exact component values.
