    **GoProChunk** — public struct, Equatable, Hashable:
    ```swift
    public struct GoProChunk: Equatable, Hashable {
        public let prefix: String       // "GH" or "GX"
        public let chapter: Int         // 1-99 (from 2-digit zero-padded string)
        public let fileNumber: Int      // 1-9999 (from 4-digit zero-padded string)

        public init(prefix: String, chapter: Int, fileNumber: Int)

        /// Reconstructed filename, e.g. "GH010001.MP4"
        public var filename: String {
            String(format: "%@%02d%04d.MP4", prefix, chapter, fileNumber)
        }
    }
    ```

    **GoProNameParser** — public enum (namespace):
    ```swift
    public enum GoProNameParser {
        /// Parse a filename like "GH010001.MP4" into a GoProChunk.
        /// Returns nil for any non-matching filename.
        public static func parse(_ filename: String) -> GoProChunk?

        /// Sort chunks by fileNumber ascending, then chapter ascending.
        /// This gives correct stitch order for multi-recording folders.
        public static func sortedChunks(_ chunks: [GoProChunk]) -> [GoProChunk]
    }
    ```

    **parse() implementation:**
    - Use a regex or manual string parsing. Manual approach: check filename.count == 12 (2 prefix + 2 chapter + 4 fileNumber + 4 extension ".MP4"), check extension == ".MP4" (uppercase only), check prefix in ["GH", "GX"], parse chapter from characters 2-3 as Int, parse fileNumber from characters 4-7 as Int. Return GoProChunk or nil.
    - Alternatively use NSRegularExpression: pattern `^(GH|GX)(\d{2})(\d{4})\.MP4$` — match group 1=prefix, group 2=chapter, group 3=fileNumber.
    - Prefer NSRegularExpression for clarity and correctness. Use `.init(pattern:)` once (static let). Handle optional Int conversion: if Int(chapterStr) or Int(fnStr) is nil, return nil.

    **sortedChunks() implementation:**
    ```swift
    chunks.sorted { a, b in
        if a.fileNumber != b.fileNumber { return a.fileNumber < b.fileNumber }
        return a.chapter < b.chapter
    }
    ```
