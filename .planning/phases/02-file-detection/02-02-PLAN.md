    **ScannedChunk** — public struct, Equatable:

    ```swift
        public let chunk: GoProChunk   // Parsed metadata
        public let url: URL            // Absolute file URL
        public let sizeBytes: Int      // File size in bytes
    }
    ```

    **FolderScanResult** — public enum:
    ```swift
    public enum FolderScanResult: Equatable {
        case success([ScannedChunk])
        case empty                     // Folder has no files at all
        case noGoProFiles              // Folder has files but none match GH/GX pattern
    }
    ```

    **FolderScanner** — public enum (namespace):
    ```swift
    public enum FolderScanner {
        public static func scan(url: URL) -> FolderScanResult
    }
    ```

    **scan() implementation:**
    1. Use FileManager.default.contentsOfDirectory(at:includingPropertiesForKeys:[.fileSizeKey], options:.skipsHiddenFiles) to list directory contents. On error (directory unreadable), return .empty.
    2. Filter for items where item.pathExtension.uppercased() == "MP4".
    3. If no .MP4 files found: check if total contents is also empty → if directory was empty return .empty, else return .noGoProFiles. (Simplest: if all files list is empty → .empty; if MP4 list is empty but files list non-empty → .noGoProFiles.)
    4. For each .MP4 URL, call GoProNameParser.parse(url.lastPathComponent). If parse returns nil, skip. If parse succeeds, read file size from resourceValues(forKeys:[.fileSizeKey]) — use (try? url.resourceValues(forKeys:[.fileSizeKey]))?.fileSize ?? 0.
    5. Collect ScannedChunk values. If resulting array is empty (all MP4s rejected by parser), return .noGoProFiles.
    6. Return .success(GoProNameParser.sortedChunks(chunks).map { ... }) — apply sort to final array. Actually: collect ScannedChunk values, then sort by chunk.fileNumber then chunk.chapter (mirrors GoProNameParser.sortedChunks logic but applied to ScannedChunk array).

    Sort ScannedChunk array:
    ```swift
    chunks.sorted { a, b in
        if a.chunk.fileNumber != b.chunk.fileNumber { return a.chunk.fileNumber < b.chunk.fileNumber }
        return a.chunk.chapter < b.chunk.chapter
    }
    ```
